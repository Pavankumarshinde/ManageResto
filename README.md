# ManageResto

**ManageResto** is a comprehensive restaurant management application designed for waiters, managers, and owners to streamline daily operations.

### Key Features:
- **Order Management**: Take, edit, and track the status of restaurant orders in real-time.
- **Menu Management**: Easily add, edit, or remove menu items to keep offerings up to date.
- **Real-Time Syncing**: Supports multiple concurrent users with synchronized updates across all devices (1-second polling).
- **Responsive Design**: Optimized for all screen sizes, from mobile phones to tablets and laptops.

### Core Modules:
- **Orders Page**: Monitor active and completed orders with status tracking for kitchen preparation and service.
- **Menu Page**: Centralized hub for managing dishes, pricing, dietary preferences, and categories.
- **Analytics Page**: Gain insights into revenue, order volume, and popular items with day/month-based views.

Built for efficiency, ManageResto ensures that when a waiter adds an order, the kitchen can see it almost instantly, improving service speed and coordination.
